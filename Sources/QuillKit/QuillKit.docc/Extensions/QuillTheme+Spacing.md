# ``QuillTheme/Spacing``

Shared spacing tokens applied between block elements.

## Topics

- ``blockSpacing``
- ``init(blockSpacing:)``

## ``blockSpacing``

Vertical spacing inserted between adjacent block elements, as a ``SpacingValue``. Relative values scale with body font size.

## ``init(blockSpacing:)``

Creates a Spacing token group.

- Parameter blockSpacing: Vertical spacing between block elements.
