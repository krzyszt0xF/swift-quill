# ``QuillTheme/Link``

Token group for link styling.

## Overview

Controls link color and underline style. Accessibility guidance recommends links be distinguishable from body text by more than color alone.

## Topics

### Properties

- ``color``
- ``underlineStyle``
- ``init(color:underlineStyle:)``

## ``color``

Foreground color for link text. Use `UIColor` dynamic providers for light/dark variants.

## ``underlineStyle``

Underline style applied to links. Use `.single` for standard underlined links.

## ``init(color:underlineStyle:)``

Creates a Link token group.

- Parameter color: Foreground color for link text.
- Parameter underlineStyle: Underline style applied to links.
