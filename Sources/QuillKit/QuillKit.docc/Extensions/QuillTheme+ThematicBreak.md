# ``QuillTheme/ThematicBreak``

Token group for horizontal rule (`---`) rendering. Controls color and vertical spacing of thematic breaks -- single-line horizontal rules separating content sections.

## Topics

- ``color``
- ``spacing``
- ``init(color:spacing:)``

## ``color``

Color of the horizontal rule line. Typically `.separator` or a comparable system semantic color.

## ``spacing``

Vertical spacing applied above and below the rule, as a ``SpacingValue``.

## ``init(color:spacing:)``

Creates a ThematicBreak token group.

- Parameter color: Color of the horizontal rule.
- Parameter spacing: Vertical spacing above and below the rule.
